## VvInput

Let's define a story for our input (default) `VvInput`:

<Canvas>
	<Story
		name="binded"
		play={async ({ args, canvasElement }) => {
			const canvas = within(canvasElement)
			await userEvent.click(canvas.getByRole('textbox'))
			await userEvent.type(
				canvas.getByRole('textbox'),
				'Hello from VvInput'
			)
		}}>
		{() => {
			return {
				components: { VvInput },
				template:
					/*html*/
					`<vv-input v-model="value" />`,
				data() {
					return {
						value: ''
					}
				}
			}
		}}
	</Story>
