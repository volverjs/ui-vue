# Rounded

A VvButtonGroup with the <strong>rounded</strong> prop will be render with <strong>.vv-button-group--rounded</strong> css class. <br/>

<Source language="jsx" code={'<vv-button rounded></vv-button>'} />

<Canvas>
	<Story
		name="rounded"
		play={(ctx) =>
			buttonGroupTest({
				...ctx,
				functions: [classTest],
				className: 'rounded'
			})
		}>
		{() => {
			return {
				components: { VvButton, VvButtonGroup },
				template: `
                        <vv-button-group modifiers="rounded">
                            <vv-button>Button 1</vv-button>
                            <vv-button id="btn2">Button 2</vv-button>
                            <vv-button>Button 3</vv-button>
                        </vv-button-group>
                    `
			}
		}}
	</Story>
</Canvas>

# Mix rounded buttons

You can manage the rounded stile of single buttons into group. Set rounded=false and add <strong>.vv-button--rounded</strong> to single buttons.
