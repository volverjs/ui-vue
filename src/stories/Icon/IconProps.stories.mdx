# VvIcon prefix

Stories of `VvIcon` `prefix`.
By default prefix generated are: `simple`, `normal`, `detailed`

## Prefix normal (default)

Let's define a story for our `normal` `prefix` icon.

<Canvas>
	<Story
		name="prefix"
		args={{
			name: 'add',
			prefix: 'normal'
		}}
		argTypes={{
			prefix: {
				control: {
					type: 'select',
					options: ['simple', 'normal', 'detailed']
				}
			}
		}}
		play={(ctx) =>
			iconTest({
				...ctx,
				functions: [classTest],
				classNames: ['vv-icon', 'iconify', 'iconify--normal']
			})
		}>
		{Template.bind({})}
	</Story>
</Canvas>
